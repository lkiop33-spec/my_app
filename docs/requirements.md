# 기능 정의서 (Requirements Specification)

## 1. 프로젝트 개요
* **프로젝트명:** 저숙련자 작업지원 플랫폼 (WSSP - Woori Smart Support Platform)
* **적용처:** 충주 인팩(INFAC) PCB(ECU, BCM 등) 생산 및 수작업 조립 라인
* **목표:** 현장에 투입된 저숙련/외국인 작업자가 스마트 단말기(AR/비전 가이드, 다국어 매뉴얼 등)를 통해 공정 오류를 최소화하고 빠르게 숙련도를 정밀 수준(목표치)까지 향상할 수 있도록 지원. 또한 관리자가 현장 전체의 가동 현황과 작업자별 성장률을 한눈에 파악할 수 있는 통계 시스템을 제공.
* **주요 타겟:** PCB 생산 라인 현장 수작업자(신입, 저숙련공, 외국인 노동자 등) 및 공정 관리자(현장 반장, 생산 관리자).

## 2. 사용자 역할 (Roles)
* **작업자 (Worker):** 부여받은 단말기(Tablet 등)를 통해 자신의 계정으로 로그인하고, 공정에 필요한 다국어 매뉴얼, 비전 박스(Bounding Box) 포지션 가이드라인을 확인하며 실시간으로 PCB를 조립하는 사용자.
* **관리자 (Admin):** 대시보드를 통해 현장 전체의 실시간 오류 발생률, 개별 작업자의 숙련도 추이(레이더/라인 차트), 단말기 상태, 공정별 BOM(자재 명세) 및 작업 지시를 통합적으로 등록, 모니터링, 통제하는 사용자.

## 4. 비기능 및 시스템 요구사항 (Non-Functional Requirements)

*   **디자인 톤앤매너:** 산업 현장에서 장시간 쳐다봐도 눈 피로가 덜한 일관적인 전체 다크 모델(Dark Mode) 및 세련된 네온 컬러(전문 측정기기 느낌) 차트 탑재.
*   **Web-UI 반응성:** 현장 태블릿 및 PC 어디에서 접속해도 깨지지 않는 Tailwind CSS 기반의 템플릿(반응형). 전면적으로 Web 상에서 CRUD 데이터 컨트롤 수행 기능 보장.
*   **보안:** URL 직접 접근 차단 기법을 포함하여 Auth/Verified 미들웨어를 통과한 유저만 접속 인가.
*   **안정성:** 개발용 마이그레이션이 아닌, 더미 데이터(Seeder)를 통해 개발단계에서 실제 테이블(13종 이상) 제약 무결성 검증 완비.
